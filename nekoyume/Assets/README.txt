Nine Chronicles Alpha
=====================

Welcome! We're very excited to have you onboard :)


Installation Steps
------------------
1. Run `NineChronclesSnapshot` (or `NineChronclesSnapshot.exe`) to download blockchain data
2. Run `Nine Chronicles` (or `Nine Chronicles.exe`) to play

You may have to give the application permission to launch. Refer to http://bit.ly/9c-alpha-launch-instructions for detailed instructions.


Troubleshooting
----------------

1. First refer to http://bit.ly/9c-alpha-launch-instructions
2. Please visit https://discord.gg/planetarium to report any problems to the developers
